# 🧸 Jomelle's Toys and Accessories
**Full-Stack E-Commerce Platform | Lead Developer & QA**

A comprehensive web application designed for a specialized toy and accessory retail business. This project demonstrates a complete e-commerce workflow, including secure user authentication, dynamic product cataloging, and a robust administrative backend for inventory management.

---

## 🚀 Project Overview
Jomelle's Toys and Accessories was built to solve the challenge of digital transformation for local retail. The platform focuses on high-performance product rendering and a seamless user journey from discovery to checkout.

* **Documentation:** 📖 [View Full User Manual](documentation/user-manual.md)

---

## ✨ Key Features
* **Dynamic Product Catalog:** Filterable categories and detailed product views for enhanced UX.
* **Secure Authentication:** Separate portals for **Customer** and **Admin** users with encrypted credential handling.
* **Order Lifecycle:** Integrated cart system, multi-step checkout, and automated digital receipts.
* **Admin Dashboard:** Full CRUD (Create, Read, Update, Delete) capabilities for managing inventory and tracking sales.
